https://github.com/MordWraith/MapKillCounter

GameHelper2 map and session kill counter: N/M/R/U rarity counts, map timer, optional session overlay, horizontal or vertical layout, resizable overlays. Timer can pause in town/hideout, on ESC pause menu, and when the game is in background. By MordWraith for the community fork. Read-only overlay — no input automation.

Build from source only (no prebuilt DLL). Clone into Plugins/MapKillCounter inside your GameHelper2 tree, then dotnet build GameHelper and the plugin. Enable it under GameHelper -> Plugins.

Compatible with Gordin GameHelper2 main, .NET 10 (net10.0-windows, x64). Tested on upstream main @ 4f6fd2a (June 2026); should work on community forks using the same API.

Feedback / feature requests: this thread. Crashes or install issues: #help with tag plugin.
